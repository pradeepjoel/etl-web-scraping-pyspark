# ETL Web Scraping Project

## Overview
End-to-end ETL pipeline that extracts book data from a public website, transforms and validates the data, and loads it into a database for analytics.

## Tech Stack
- Python
- Pandas
- BeautifulSoup
- SQL (SQLite)
- Git

## Pipeline Steps
1. Extract data via web scraping
2. Transform and validate raw data
3. Load clean data into database

## How to Run
python src/pipeline.py
